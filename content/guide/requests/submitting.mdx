## Submitting a request

1. Go to **Requests** from the sidebar
2. Click **Add Request**
3. Fill in the details:

| Field | Description |
|-------|-------------|
| **Title** | Brief description of the problem |
| **Category** | Device Issue, Equipment Issue, or Facility Maintenance |
| **Urgency** | Low, Medium, High, or Critical |
| **Description** | Additional context — what happened, when it started, what you've tried |
| **Room** | Where the issue is located (optional) |
| **Equipment** | Related equipment, if applicable (optional) |
| **Location description** | Free text for specifics like "front desk printer" or "Operatory 3 chair" (optional) |

4. Click **Submit Request**

## Choosing the right urgency

| Urgency | When to use |
|---------|-------------|
| **Low** | Minor issue, doesn't affect patient care or daily operations |
| **Medium** | Standard priority — should be addressed soon but not immediately |
| **High** | Significant impact on operations or patient care |
| **Critical** | Blocks practice functionality — needs immediate attention |

## What happens after you submit

- Your request appears in the **Submitted** column on the Requests board
- All practice admins receive an email notification and an in-app alert
- You'll be notified when an admin updates the status of your request
