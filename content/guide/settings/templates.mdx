## What are templates?

Templates let you create reusable task definitions — standard checklists, procedures, and compliance workflows that you use repeatedly. Instead of creating the same task from scratch each time, start from a template.

## Using templates

Templates are available when creating new tasks. Select a template to pre-fill the task title, description, type, category, and assignment.

## Managing templates

Practice admins can create and edit templates from **Settings > Templates**.
