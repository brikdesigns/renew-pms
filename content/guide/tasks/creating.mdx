## Creating a new task

1. Go to **Tasks**
2. Click **New Task**
3. Fill in the details:

| Field | Description |
|-------|-----------|
| **Title** | What needs to be done |
| **Description** | Additional details or instructions |
| **Type** | Checklist, Procedure, Compliance, Skill Training, Onboarding, or Request |
| **Category** | Cleaning, Equipment, Maintenance, Compliance/Safety, Patient Care, Training, or Administrative |
| **Priority** | Low, Medium, High, or Critical |
| **Assigned to** | A specific team member (optional) |
| **Department** | Assign to a whole department instead (optional) |
| **Room** | Where this task happens (optional) |
| **Equipment** | Related equipment (optional) |
| **Due date** | When it should be completed |
| **Frequency** | One-time, daily, weekly, monthly, etc. |

## Recurring tasks

Set the **Frequency** field to automatically create repeating tasks. Options include daily, weekly, bi-weekly, monthly, quarterly, semi-annually, annually, or per shift.

## Compliance tasks

For compliance-related work (OSHA, HIPAA, etc.), set the **Type** to Compliance and select the relevant **Compliance Type**. This ensures proper tracking and reporting.
