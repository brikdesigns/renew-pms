What's new in Renew, newest first. Renew is in **Beta** — we're shipping updates regularly while we refine the product with our early practices. You can always see the version you're on in the user menu (bottom-left), shown as `Beta · v0.3.0`.

## v0.3.0 — June 2, 2026

### New

- **See your version at a glance** — The user menu (bottom-left) now shows which version of Renew you're on, like `Beta · v0.3.0`. Handy when you want to mention it while contacting support.

---

## v0.2.0 — June 2, 2026

### New

- **Recurring tasks** — Tasks can now repeat on a **weekly** or **monthly** schedule. Routine work shows up on the board automatically, so nobody has to re-create the same tasks each cycle.
- **Per-template reset cadence** — In template settings, choose how often each task template resets to match your practice's rhythm.
- **Due-date chips on task cards** — Task cards now show a due-date chip at a glance, making it easier to see what's coming up.

### Fixed

- **Feedback widget** — Feedback you submit now reliably reaches our team, and the widget tells you if something goes wrong instead of failing silently.
