- [Database Tables](/docs/schema/tables) — complete table reference with columns and relationships
- [Enum Fields](/docs/schema/enums) — fields where app logic depends on the value (not user-renameable)
- [Row Level Security](/docs/schema/rls) — RLS policies, helper functions, and common patterns
