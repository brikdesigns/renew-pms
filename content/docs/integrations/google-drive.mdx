## Overview

Google Drive serves as the document source of truth for practice files, SOPs, and reference documents. The integration surfaces relevant docs in task and workflow context.

## Configuration

The folder ID is stored per-practice in the `integrations` JSONB column:

```json
{
  "gdrive": {
    "folder_id": "1abc..."
  }
}
```

This lives in `practices.integrations` and is **server-side only**.
